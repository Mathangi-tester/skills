---
name: Test Case Generator
description: Generate test cases from requirement files and export them into Excel format.
---

## When to use

Use this skill when:
- A requirement document is provided.
- A text file contains functional requirements.
- Test cases need to be generated and exported to Excel.

## Instructions

1. Read the requirement file.
2. Identify each requirement.
3. Create:
   - Positive test cases
   - Negative test cases
   - Boundary test cases
   - Validation test cases

4. Populate:

- Test Case ID
- Requirement ID
- Test Scenario
- Test Steps
- Expected Result
- Priority
- Test Type

5. Produce output in tabular format suitable for Excel.
